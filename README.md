# 🔐 Gerador de Senhas

Projeto desenvolvido em HTML, CSS e JavaScript.

## Funcionalidades

- Gera senhas aleatórias.
- Escolha do tamanho da senha.
- Letras maiúsculas e minúsculas.
- Números.
- Caracteres especiais.
- Botão para copiar a senha.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Objetivo

Este projeto foi desenvolvido para praticar programação e conceitos de segurança digital, criando senhas fortes e aleatórias.
